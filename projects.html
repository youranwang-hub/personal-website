基于 STM32F407 的嵌入式综合项目，系统完成 22 个实验，串联 12 个外设模块，模拟冷链运输全流程。

系统性完成 22 个实验，分五阶段递进：基础外设（GPIO / 中断 / USART / TIM / PWM / ADC / DAC / DMA / RTC）、传感器（DHT11 / MQ2）、总线与协议（I2C / SPI / CAN / UART-GPS）、人机交互（LCD / 触摸屏 / 红外 NEC）、综合应用（ZN632 指纹 / PN532 NFC）。

最终将所有模块整合为智能物流配送终端：指纹或触摸屏右滑认证司机身份，GPS 定位，DHT11 与 MQ2 循环监控货箱环境并超限触发蜂鸣器报警，红外遥控模拟收货签收并写入 FLASH 存档，QCOM 串口菜单调度，LCD 同步显示关键状态。串联 12 个外设模块，呈现从驱动开发到场景落地的工程实践全貌。
